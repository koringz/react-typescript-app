# react-typescript-app

react-typescript-app 脚手架

## 支持功能

typescript, react, scss, less, plop, antd

## NodeJS

v13.14.0

## 初次打开

> yarn open

## 后续运行

> yarn go

## 模板构建

> yarn plop

## 依赖包

```bash
npm config set registry=https://registry.npmjs.org/
```

wait... 输入你的组件名 {{ComponentName}}
